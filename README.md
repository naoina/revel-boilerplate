# Revel Boilerplate

A Boilerplate for [Revel](https://github.com/robfig/revel) application.

## Usage

### Development

* git clone https://github.com/naoina/revel-boilerplate *your_app_name*
* cd *your_app_name*
* npm install
* bower install
* grunt
* revel run *your_app_name*

### Production
* grunt build
* revel run *your_app_name* prod
